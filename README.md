# Machine Learning Repository

A comprehensive collection of Python implementations for both **Supervised** and **Unsupervised Machine Learning** algorithms. This repository contains educational notebooks, scripts, and datasets for learning and practicing various machine learning techniques.

## Project Structure

```
Supervised-Machine-Learning/
├── Supervised-Learning/          # Supervised learning algorithms
├── Unsupervised-Learning/        # Unsupervised learning algorithms
├── Datasets/                     # Training and testing datasets
└── README.md
```
